  
## Sample Data Set of Videos
Multimodal Dataset for Sentiment Analysis and Classification, published by the same contributor, which also contains 600 social media videos categorized into the seven emotion classes mentioned above.

https://data.mendeley.com/datasets/b7z68nykxt/1

Total Videos: 600  

Content: Social media videos sourced from platforms including Instagram, YouTube, TikTok, Facebook, Twitter, and Telegram.

Labeled Emotions: Neutral, Happy, Anger, Disgust, Fear, Sadness, and Surprised.  

Format: The dataset is described as balanced, featuring US English speakers of varying ages and ethnicities, with each clip containing a single speaker.
